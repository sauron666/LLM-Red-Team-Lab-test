# Security Policy

## Scope
LLM Attack Lab is a security testing framework for authorized use only.

## Reporting a vulnerability
If you discover a vulnerability in the tool itself:
1. Do not open a public issue with exploit details.
2. Send a private report to the maintainer (or use your internal channel if this is a private fork).
3. Include reproduction steps, impact, affected version, and logs/screenshots.

## Safe operation guidelines
- Use only on systems you own or are authorized to test.
- Use safe mode by default.
- Burp replay strips auth headers by default.
- Redact secrets from reports before sharing externally.
